Engineered an intuitive, effective way to manage and monitor temperature profiles for PCB soldering by constructing a controller system using an ARMv7 processor and an thermocouple, integrating an LCD and hardware remote. An engaging Python app interface was developed to allow users to input soldering settings and graph temperatures dynamically.
